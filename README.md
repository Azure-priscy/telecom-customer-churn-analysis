# telecom-customer-churn-analysis

## Project Overview
This project explores telecom customer behavior, billing patterns, subscribed services, and customer churn tendencies using exploratory data analysis techniques.

The purpose is to uncover hidden patterns that influence customer dissatisfaction, retention, and service loyalty.

---

## Project Objectives
- Analyze customer demographics and telecom account information
- Examine billing and payment behavior
- Investigate service subscription patterns
- Explore customer churn distribution
- Identify possible factors associated with customer loss
- Generate business recommendations from insights

---

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook / Google Colab

---

## Dataset Information
The dataset contains 7,043 telecom customer records with 21 structured variables covering:
- customer tenure
- monthly charges
- payment methods
- internet services
- support services
- contract type
- churn status
